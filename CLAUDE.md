# LogisticsInventorySystem

Property management and logistics inventory platform with .NET Core API and Blazor frontend.

## Projects
- LogisticsAPI/ — ASP.NET Core 8.0 Web API
- LogisticsUI/ — Blazor Server-Side frontend
- LogisticsAPI.Tests/ — xUnit test project

## Running locally
dotnet run --project LogisticsAPI
dotnet run --project LogisticsUI

## Running tests
dotnet test
